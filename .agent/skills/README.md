# Antigravity Skills (Organized)

This directory contains specialized AI agent skills, neatly categorized to provide structured, progressive disclosure of knowledge. The categorizations help maintain context cleanly without overwhelming the agent.

## Directory Structure

- **`01_Core_Architecture/`**: Clean code principles, code review checklists, project planning, architecture patterns, and brainstorming guides.
- **`02_Frontend/`**: Frontend design systems, Tailwind patterns, Next.js/React expert rules, web & mobile design guidelines.
- **`03_Backend_Data/`**: API design patterns (REST, GraphQL, tRPC), Node.js and Python backend best practices, database design, and Rust workflows.
- **`04_DevOps_Deployment/`**: Server management, deployment procedures, Shell (Bash/Powershell) scripting best practices.
- **`05_Quality_Testing/`**: Systematic debugging, TDD workflows, testing patterns, profiling, and linting standards.
- **`06_Security/`**: Vulnerability scanning, red team tactics, and security testing principles.
- **`07_AI_Agents/`**: Guidelines for MCP building, parallel agents, prompt engineering (enhancer), intelligent routing, and feature advising.
- **`08_Specialized/`**: Specialized tasks like game development, SEO fundamentals, i18n localization, app-builder instructions, documentation templates, agent-repo-init (scaffolding new multi-agent projects), and deep research protocols.
- **`09_Scripts/`**: Internal scripts used by the agent for automation or validation within skills.

> **Agent Note**: You may navigate into any of these categories and read the `SKILL.md` inside any specific skill subfolder when required by your current task.
