# 🚀 Ag-Ultimate-Kit (Master Agent Workspace)

**The Ultimate, fully autonomous AI Agent integration kit.**

This repository is a master culmination of the most advanced AI agent behaviors, rules, and workflows currently available in the industry (borrowing architectures from _Cursor_, _Claude Code_, _Windsurf_, and deep repositories like _study8677_, _vudovn_, and _omergocmen_).

If you are using an AI Agent IDE (like Antigravity, Cursor, or Claude), integrating this `.agent` folder into your project will instantly upgrade your AI assistant from a "chatbot" to an **autonomous senior developer and business execution engine**.

---

## 🌟 Key Features

1. **🧠 Advanced AI Rules (`.agent/rules`)**
   - Implements Socratic questioning, "Root-Cause Debugging", and "Chunk Edits" from Windsurf.
   - Enforces the **Shout Protocol**: The agent will explicitly warn you before modifying deeply integrated legacy code.
   - "Trace to Source" behavior ensuring the agent never hallucinates variables.

2. **🛠️ 50+ Specialized Skills (`.agent/skills`)**
   - Cleanly organized into 9 cognitive lobes (e.g., `01_Core_Architecture`, `02_Frontend`, `06_Security`, `09_Scripts`).
   - Uses _Progressive Disclosure_ so the agent only loads the skill it needs, saving context window and money.

3. **📜 14 Robust Workflows (`.agent/workflows`)**
   - Includes world-class 11KB+ `ui-ux-pro-max.md` workflow for pixel-perfect frontend generation.
   - Adopts the **OpenSpec Architecture** (`openspec-proposal`, `openspec-apply`, `openspec-archive`) allowing AI to scaffold enterprise changes safely.
   - `orchestrate.md`: Script to choreograph multiple AI agents passing tasks back and forth.

4. **🤖 Agentic Autonomy (`AGENTS.md` & Directives)**
   - Includes an execution layer for deterministic Python scripts.
   - Teaches the AI the **Self-Annealing Loop**: If an error occurs, the AI reads the stack trace, fixes the script, tests it, and updates its own operational directives automatically.

## ⚡ Quick Start

To empower your next project with this mastermind AI, simply copy the `.agent` folder into the root of your target project.

### 🍎 Mac / Linux (Bash)

```bash
# 1. Clone the master kit
git clone https://github.com/smhaydn/Ag-Ultimate-Kit.git

# 2. Copy the .agent folder to your target project folder
cp -r Ag-Ultimate-Kit/.agent /path/to/your/target/project/
```

### 🪟 Windows (PowerShell)

```powershell
# 1. Clone the master kit
git clone https://github.com/smhaydn/Ag-Ultimate-Kit.git

# 2. Copy the .agent folder to your target project folder
Copy-Item -Path "Ag-Ultimate-Kit\.agent" -Destination "C:\path\to\your\target\project\.agent" -Recurse -Force
```

## 🏗️ Manual Installation

If you prefer to integrate manually:

1. Download or clone this repository.
2. Locate the `.agent` folder.
3. Drag and drop the `.agent` folder into the root directory of your own project.
4. Open your project in your AI IDE (Antigravity, Cursor, etc.).
5. The AI will automatically detect the `.agent/rules/General.md` and `.agent/workflows` and instantly adopt the new behavior framework.

---

## 💡 How to Use Once Installed

Once the `.agent` folder is in your project, you don't need to do anything complex. Just talk to your AI naturally.

- **To start a new feature:** Ask the AI to `/plan` or "Run the plan workflow". It will read `.agent/workflows/plan.md` and act accordingly.
- **For frontend tasks:** Ask the AI to "Design a login page using ui-ux-pro-max workflow."
- **For special coding help:** "I need to set up a Node backend, check your skills for backend patterns." The AI will navigate to `.agent/skills/03_Backend_Data` and learn the best practices before writing code.
